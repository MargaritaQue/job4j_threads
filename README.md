В разделе "Multithreading" мы познакомимся с многопоточностью в Java. 
Многопоточные приложения намного производительные, чем однопоточные, 
так как позволяют выполнять несколько задач одновременно, то есть делить 
задачи между потоками (нитям). Многопоточность - это сложный механизм, 
незнание которого может привести к различным негативным последствиям при 
работе в многопоточной среде.
